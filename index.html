ⓘ 关于 KM

关于本站

KM (空门) 是一个个人能力关系图谱网站，旨在以可视化的方式展示技能结构与知识脉络。 通过力导向图直观呈现各项能力之间的关联，帮助梳理个人技术栈。

技术栈

HTML / CSS JavaScript D3.js GitHub Pages

说明

站点采用纯前端实现，所有数据存储在本地。能力图谱中的节点大小代表熟练程度， 连线代表技能之间的关联关系。点击节点可查看详细信息。
